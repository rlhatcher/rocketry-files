# Rocketry Files

Designs, simulations, build notes and flight data for a collection of model and
high-power rockets — OpenRocket sims, printable CAD (STL / 3MF / STEP), and where
available recorded flight data. Open a project folder above, or use search.

Each project page lists its files with inline previews: 👁 opens a 3D view for
`.3mf` / `.stl` / `.step` models (and images), ⬇ downloads. CAD/editor sources are
kept in Git LFS and link out to GitHub.

**Highlight —** the [Macie XL flight dashboard](macie/flights/MRC%20Summer%2026/maciexl-dashboard.html)
compares OpenRocket simulation against actual flight hardware across both Macie flights.
