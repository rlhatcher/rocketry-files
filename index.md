# Rocketry Files

Designs, simulations, build notes and flight data for a collection of model and
high-power rockets. Most projects include OpenRocket simulations, printable CAD
(STL / 3MF / STEP), and where available, recorded flight data.

Use the search box at the top to find anything across the site, or browse the
projects below.

---

## ⭐ Featured — Macie XL flight dashboard

An interactive dashboard comparing the OpenRocket **simulation against actual
flight hardware** across both Macie XL flights — altitude, velocity, acceleration,
plus EasyTimer orientation data — with event markers and a configurable staging
tilt guide.
